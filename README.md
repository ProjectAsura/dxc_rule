# dxc_rule
Custom Build Rule for DXC

## 使い方
Visual Studio上のソリューションエクスプローラーで右クリックし，「ビルドの依存関係」>「ビルドのカスタマイズ」を選択します。  
Visual C++ビルドカスタマイズファイルという名前のダイアログが表示されるので，「既存ファイルの検索」を選択し，dxc.targetsを指定してください。
指定後に「dxc」にチェックボックスが入っていることを確認してください。  

*.hlslファイルを作成し，ソリューションエクスプローラー上から*.hlslを選択し，右クリックでプロパティを表示させます。
プロパティウィンドウの「構成プロパティ」> 「全般」を選択し，「項目の種類」を「DXC コンパイラ」に変更してください。
あとは，「DXC コンパイラ」のプロパティからシェーダモデルや，シェーダプロファイル，エントリーポイント名を指定して，ビルドまたはコンパイルを実行してください。
